# 2\. 学习顺序 (Learning Order)

| **Type**<br><br>**类型**                    | **Core Idea**<br><br>**核心概念**                                      | **How it works**<br><br>**运作方式**                                      | **Usage / Example**<br><br>**用途 / 示例**                              |
| ------------------------------------------- | ---------------------------------------------------------------------- | ------------------------------------------------------------------------- | ----------------------------------------------------------------------- |
| **Foundation First**<br><br>**先基础**      | Always learn basic concepts before advanced topics <br>先学基础再进阶  | Follow order: variables to functions gradually <br>按照变量到函数逐步学习 | Build solid understanding of programming basics <br>打好编程基础         |

| **Concept → Practice**<br><br>**概念→练习** | Apply concepts immediately after learning <br>学完马上进行实践         | Study example, modify it, then create own <br>先看例子再修改最后自己写     | Strengthen understanding through direct application <br>通过实践强化理解 |

| **Type**<br><br>**类型**                       | **Core Idea**<br><br>**核心概念**                                      | **How it works**<br><br>**运作方式**                                      | **Usage / Example**<br><br>**用途 / 示例**                              |

# 4\. Debug 的方法 (How to Debug)

| **Type**<br><br>**类型**              | **Core Idea**<br><br>**核心概念**                                      | **How it works**<br><br>**运作方式**                                      | **Usage / Example**<br><br>**用途 / 示例**                              |
| ------------------------------------- | ---------------------------------------------------------------------- | ------------------------------------------------------------------------- | ----------------------------------------------------------------------- |
| **Read Errors**<br><br>**读错误信息** | Error messages clearly describe the issue <br>错误信息直接说明问题      | Check line number and error description <br>查看错误行号和提示内容         | Fix problems quickly and accurately <br>快速准确定位问题                 |
| **Step-by-Step**<br><br>**检查步骤**  | Examine code logic line by line carefully <br>逐行检查代码逻辑          | Track variable values and execution flow <br>追踪变量变化和执行过程        | Identify hidden logical mistakes easily <br>发现隐藏的逻辑错误           |
| **Use print()**<br><br>**调试**       | Print variables to observe program state <br>打印变量观察程序状态       | Add print before and after key steps <br>在关键步骤前后打印变量            | Understand how program flows internally <br>理解程序执行流程             |
| **Simplify Code**<br><br>**简化代码** | Reduce code to isolate the problem <br>简化代码以定位问题               | Remove unrelated parts and focus core logic <br>删除无关代码保留核心部分   | Make debugging process faster and clearer <br>让调试更清晰高效           |
| **Common Errors**<br><br>**常见错误** | Many bugs come from simple mistakes <br>很多错误来自简单问题            | Check syntax, spelling, and indentation <br>检查拼写语法和缩进             | Learn patterns of common mistakes quickly <br>快速识别常见错误类型       |
