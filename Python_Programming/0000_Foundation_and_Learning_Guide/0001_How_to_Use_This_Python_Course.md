# 2\. 学习顺序 (Learning Order)

| **Type**<br><br>**类型**                    | **Core Idea**<br><br>**核心概念**                                | **How it works**<br><br>**运作方式**                               | **Usage / Example**<br><br>**用途 / 示例**  |
| ------------------------------------------- | ---------------------------------------------------------------- | ------------------------------------------------------------------ | ------------------------------------------- |
| **Foundation First**<br><br>**先基础**      | Start with basics before advanced topics <br>先学基础再学进阶    | Variables → Conditions → Loops → Functions <br>变量→判断→循环→函数 | Strong core understanding <br>打好基础      |
| **Do Not Skip**<br><br>**不跳课**           | Every topic is connected <br>每个知识点是连接的                  | Skipping leads to confusion later <br>跳过会导致后面不懂           | Prevents gaps in knowledge <br>避免知识断层 |
| **Layered Learning**<br><br>**分层学习**    | Learn in levels (basic → intermediate → advanced) <br>分阶段学习 | Build complexity gradually <br>逐步增加难度                        | Better understanding <br>更容易理解         |
| **Concept → Practice**<br><br>**概念→练习** | Learn concept then apply immediately <br>学完马上练              | Example → Modify → Create own code <br>看例子→修改→自己写          | Reinforces learning <br>强化理解            |

| **Type**<br><br>**类型**                       | **Core Idea**<br><br>**核心概念**                         | **How it works**<br><br>**运作方式**                    | **Usage / Example**<br><br>**用途 / 示例** |

# 4\. Debug 的方法 (How to Debug)

| **Type**<br><br>**类型**              | **Core Idea**<br><br>**核心概念**                        | **How it works**<br><br>**运作方式**                 | **Usage / Example**<br><br>**用途 / 示例** |
| ------------------------------------- | -------------------------------------------------------- | ---------------------------------------------------- | ------------------------------------------ |
| **Read Errors**<br><br>**读错误信息** | Error messages tell you the problem <br>错误信息就是答案 | Look at line number and message <br>看行号和错误类型 | Faster fixing <br>快速修复                 |
| **Step-by-Step**<br><br>**检查步骤**  | Check code line by line <br>逐行检查                     | Trace values and logic <br>追踪变量和逻辑            | Find logic errors <br>发现逻辑错误         |
| **Use print()**<br><br>**调试**       | Print values to check state <br>打印变量检查状态         | Print before and after changes <br>修改前后打印      | Understand flow <br>理解执行流程           |
| **Simplify Code**<br><br>**简化代码** | Reduce code to isolate problem <br>简化代码找问题        | Remove unnecessary parts <br>删除不相关部分          | Easier debugging <br>更容易找错            |
| **Common Errors**<br><br>**常见错误** | Many bugs are simple mistakes <br>很多错误很简单         | Typo, wrong indentation <br>拼写错误、缩进错误       | Recognize patterns <br>识别常见错误        |
