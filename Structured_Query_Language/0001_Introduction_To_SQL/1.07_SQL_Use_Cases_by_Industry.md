## 1.7.2.4 Transaction Systems

Examples:

| Transaction Type | Example |
|---|---|
| Money transfer | Bank transfer |
| Payment processing | Credit card payment |
| ATM withdrawal | Cash withdrawal |

---

### 1.7.2.4.2 SQL Transactions in Finance

SQL transactions ensure:

- Money is not lost
- Operations remain consistent

---

### 1.7.2.4.3 Example Transaction Workflow

| Step | Action |
|---|---|
| Step 1 | Deduct sender balance |
| Step 2 | Add receiver balance |
| Step 3 | COMMIT transaction |

If one step fails:

- ROLLBACK occurs

---

### 1.7.2.4.4 Example SQL Transaction

#### Example

```sql
BEGIN;

UPDATE accounts
SET balance = balance - 500
WHERE account_id = 1;

UPDATE accounts
SET balance = balance + 500
WHERE account_id = 2;

COMMIT;
```
