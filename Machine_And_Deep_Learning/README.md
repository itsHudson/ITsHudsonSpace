# Core Category Comparison

| Category | Description | How It Works | Real Example | Tools / Software | Relationship |
| --- | --- | --- | --- | --- | --- |
| **Computer Science (CS)** | Write programs that follow exact instructions | 1. Write code → <br><br> 2. Computer executes exactly as written → <br><br> 3. No learning occurs | Sorting numbers `[5,2,9] → [2,5,9]` | Python, Java, C++, VS Code, Git | Largest category |
| **Artificial Intelligence (AI)** | Make computers choose actions using rules or logic | 1. Define rules → <br><br> 2. Input enters system → <br><br> 3. Rules checked → <br><br> 4. Decision/output generated | Rule-based chatbot: IF user says `"hello"` → reply `"hi"` | Python, Prolog, Rule-based systems | Subset of Computer Science |
| **Machine Learning (ML)** | Make computers learn from data to make predictions | 1. Collect training data → <br><br> 2. Train model → <br><br> 3. Find patterns → <br><br> 4. Predict new data | Spam classifier: predict spam or not spam | Python, Scikit-learn, Pandas, NumPy | Subset of Artificial Intelligence |
| **Deep Learning (DL)** | Make computers learn using neural networks with many layers | 1. Prepare large dataset → <br><br> 2. Build neural network → <br><br> 3. Train model → <br><br> 4. Automatically learn features | Image recognition: input image → output `"cat"` | Python, TensorFlow, PyTorch, GPU (CUDA) | Subset of Machine Learning |
