# Computer Science (CS)

**What it does**
Write programs that follow exact instructions

**How it works**
1. Write code  
2. Computer runs exactly as written  
3. No learning or change  

**Specific Example**
Sort numbers [5,2,9] → [2,5,9] using sorting code  

**Tools / Software**
Python, Java, C++, Visual Studio Code, Git  

**Relationship**
Largest category (not a subset of others)

---

# Artificial Intelligence (AI)

**What it does**
Make computer choose an action (decision) using rules or logic  

**How it works**
1. Define rules  
2. Input comes in  
3. System checks rules  
4. Output decision  

**Specific Example**
Rule-based chatbot: IF user says "hello" → reply "hi"  

**Tools / Software**
Python, Prolog, rule-based systems  

**Relationship**
Subset of CS

---

# Machine Learning (ML)

**What it does**
Make computer learn from data to predict output  

**How it works**
1. Collect data (input + correct answer)  
2. Train model  
3. Model finds pattern  
4. Use model to predict new data  

**Specific Example**
Spam classifier: Train with emails → predict spam or not  

**Tools / Software**
Python, Scikit-learn, Pandas, NumPy  

**Relationship**
Subset of AI

---

# Deep Learning (DL)

**What it does**
Make computer learn using neural networks (many layers)  

**How it works**
1. Prepare large dataset (images/text)  
2. Build neural network  
3. Train model  
4. Model learns features automatically  

**Specific Example**
Image recognition: Input image → output "cat"  

**Tools / Software**
Python, TensorFlow, PyTorch, GPU (CUDA)  

**Relationship**
Subset of ML
